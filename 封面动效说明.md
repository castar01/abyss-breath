# 封面动效说明

## 🌊 实现的效果

### 1. 呼吸感动画
- **效果**：封面图片缓慢放大缩小
- **周期**：6秒一个循环
- **幅度**：100% → 105% → 100%
- **感觉**：像生物在深海中呼吸

### 2. 深海波浪感
- **效果**：轻微的色相和亮度变化
- **周期**：8秒一个循环
- **变化**：
  - 亮度：95% ~ 110%
  - 色相：-2° ~ +2°
- **感觉**：水下光线的波动

### 3. 水纹叠加层
- **效果**：半透明的蓝色渐变波纹
- **周期**：10秒缓慢移动
- **颜色**：深蓝色（rgba 0, 100-150, 150-200）
- **感觉**：水流的轻微扰动

### 4. 深海光线
- **效果**：斜向移动的光线
- **周期**：12秒完整旋转
- **颜色**：青色半透明
- **感觉**：从水面透下来的微光

---

## 🎨 视觉效果组合

所有动画叠加产生的效果：
```
封面图片
├── 呼吸缩放 (6秒循环)
├── 波浪扭曲 (8秒循环)
└── 水纹层
    ├── 渐变波纹 (10秒循环)
    └── 旋转光线 (12秒循环)
```

由于周期不同，动画永不重复，产生自然的深海感觉。

## 🔧 技术实现

### CSS动画
- `deepBreathing` - 呼吸缩放
- `underwaterWave` - 波浪扭曲
- `waterFlow` - 水纹流动
- `lightRays` - 光线旋转

### 触发时机
- 游戏启动时自动播放
- 点击"开始游戏"后动效停止
- 只在封面显示，其他场景无动效

## ⚙️ 如何调整

### 加快/减慢呼吸
修改 `style.css` 第X行：
```css
animation: deepBreathing 6s ease-in-out infinite;
/* 改为 4s 会更快，8s 会更慢 */
```

### 调整呼吸幅度
修改 `@keyframes deepBreathing` 的 `scale` 值：
```css
50% {
    transform: scale(1.05); /* 1.03更轻微，1.08更明显 */
}
```

### 改变水纹颜色
修改 `#water-overlay` 的颜色值：
```css
rgba(0, 100, 150, 0.15) /* RGB和透明度 */
```

### 关闭某个效果
删除对应的动画名称即可。
