## 🎮 如何测试

### 方法1：本地测试（最简单）

1. 下载并安装 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VSCode插件

2. 在VSCode中打开`深渊呼吸`文件夹

3. 右键点击 `index.html` → 选择 "Open with Live Server"

4. 浏览器自动打开游戏

### 方法2：Python服务器

```bash
# 进入游戏目录
cd "深渊呼吸"

# 启动服务器
python3 -m http.server 8000

# 打开浏览器访问
# http://localhost:8000
```

### 方法3：直接双击

直接双击 `index.html` 文件（可能部分视频无法播放）

## 📱 移动端测试

1. 确保电脑和手机在同一WiFi
2. 查看电脑IP地址（如 192.168.1.100）
3. 手机浏览器访问：`http://192.168.1.100:8000`

## 🎯 核心交互点

### 氧气滑块（第五章-场景2）

- 拖动范围：0-50
- 实时显示：玩家剩余氧气、黑鲨状态
- **阈值判定**：
  - ≤30：分支A（失败结局）
  - >30：分支B（继续第六章）
- **数据记录**：自动保存最终数值

## 📊 数据收集

游戏自动记录以下数据：

```javascript
{
  sessionId: "TS时间戳",      // 会话ID
  groupType: "实验组/对照组",  // 随机分组
  startTime: 时间戳,
  choices: [...],             // 所有选择记录
  oxygenValue: 0-50,         // 氧气滑块值
  playTime: 秒数,             // 游戏时长
  ending: "结局类型"          // 最终结局
}
```

**数据查看位置**：
- 浏览器控制台（F12）
- 游戏结束弹窗

## 🚀 部署上线

### 方法1：Vercel（推荐）⭐

1. 注册 [Vercel](https://vercel.com)
2. 点击 "New Project"
3. 拖拽整个 `深渊呼吸` 文件夹
4. 点击 "Deploy"
5. 等待30秒，获得链接

### 方法2：Netlify

1. 注册 [Netlify](https://netlify.com)
2. 拖拽文件夹到部署区
3. 自动生成链接

### 方法3：GitHub Pages

1. 上传到GitHub仓库
2. Settings → Pages
3. 选择分支部署
4. 获得 `https://用户名.github.io/仓库名/`

## 🐛 常见问题

### 视频无法播放？
- 检查视频格式（推荐MP4 H.264编码）
- 使用Live Server或本地服务器测试

### 移动端视频自动播放失败？
- iOS需要用户交互才能播放
- 已添加 `playsinline` 属性支持

### 字体未生效？
- 需要联网加载思源黑体
- 或下载字体文件本地部署
