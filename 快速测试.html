📝 启动步骤（3选1）

方法1：命令行启动（推荐）

1. 打开终端（Terminal）

2. 输入以下命令：

cd "/Users/satellite/深渊呼吸" && python3 -m http.server 8000

3. 浏览器访问：http://localhost:8000

方法2：双击启动脚本

1. 双击 启动游戏.command 文件

2. 浏览器访问：http://localhost:8000

方法3：VSCode Live Server

1. 用VSCode打开游戏文件夹

2. 右键 index.html → Open with Live Server

🚀 直接启动游戏（可能无法播放视频）

⚠️ 常见问题

• 如果一片黑：需要用服务器启动（方法1或2）

• 视频无法播放：检查素材路径或使用Chrome浏览器

• 移动端测试：电脑启动服务器后，手机访问电脑IP
